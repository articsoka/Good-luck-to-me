HTML-The basic structure behind a web page

When opening a web page the content of the HTML instructs the browser how to display the document to the viewer.
The content is achieved throug using markup language, creating heading, paragraphs, tables, buttons, links, images and so on
Expessions you will come across learning HTML markup are for example:

  • Elements
  • Tags
  • Attributes

A Markup language

HTML is a markup language, stands for Hyper Text Markup Language, which indicates that it contains of codes that is not visible when
presented to the user. When you format text you need to distinguish between the text itself and the instructions for the text. "The markup".


HTML with CSS and JavaScript

While HTML builds the structure of a web page, CSS handles the design and
layout and the JavaScript makes the HTML page more dynamic and interactive. CSS is not a markup language, rather a style sheet language. Javascript is a
programming language.
